📄 Auto Code Documentation Generator
An AI-powered web application that automatically extracts functions from source code and generates structured documentation. This tool helps developers quickly understand code structure and improve productivity.
🚀 Features
🔍 Function Extraction – Automatically detects functions from pasted source code.
🧠 AI-powered processing for better code analysis.
📑 Automatic documentation generation for extracted functions.
🌐 Web-based interface for easy use.
⚡ Fast processing with a lightweight backend.
